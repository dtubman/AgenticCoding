# FaceClicker

A Wack-a-mole type game where all of the faces start out as seven pink squares with a smiley face. Every once in a while one of them will turn into a pink diamond with a unhappy face which the player has a few seconds to click in which case they will go back to square smiley faces. There is a 60 second timer with the time before each face getting mad decreasing progressively. If the player misses the face, then the face with be agnry for a short amount of time and go back to normal. After the 60 seconds, the player is given a summary screen.
